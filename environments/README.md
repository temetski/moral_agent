# Environments

We collected simulation environments that can showcase ethical decision-making, which involve temporality, as well as higher dimensional state-action spaces.
These simulation environments are modified from other research papers, which we will link to here:

| Environment | Source Repository |
| --- | --- |
| `FindMilk`, `Driving` | https://github.com/kristery/EthicsShaping/tree/master |

Where possible, we converted these environments to work with [`gymnasium`](https://gymnasium.farama.org/index.html).
